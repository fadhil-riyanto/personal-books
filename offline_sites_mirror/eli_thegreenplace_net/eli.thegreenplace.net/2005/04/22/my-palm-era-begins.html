my Palm era begins

April 22, 2005 at 11:50 Tags Software & Tools
Yesterday my new Palm Tungsten E arrived. Yipee ! So far it's great. The display is fantastic, and reading texts (in both Palmreader and Adobereader) is very convenient - which is my main goal for this gadget. There's a native support for European languages, so Spanish books are shown without problems. A few things left to do:
  • Buy a SD memory card for MP3s
  • Set up Cyrillic fonts to read Russian books - this shouldn't be hard to find for free, I hope.
  • Find a Spanish -> English (or Hebrew/Russian) dictionary that will be convenient to use while reading. I'm not so sure it can be found for free...
